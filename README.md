# Game AI Companion 🎮

Première base Android pour le projet de Valentin : une appli mobile pensée pour apprendre en observant le jeu, puis automatiser certaines tâches répétitives.

## Ce premier build contient
- une base Android native en Kotlin
- une interface de départ
- une configuration Gradle simple
- une GitHub Action qui génère un APK debug automatiquement

## Vision produit
À terme, l'app devra permettre :
- l'observation du jeu à l'écran
- l'enregistrement de sessions d'apprentissage
- l'analyse visuelle de l'écran
- l'exécution d'actions automatisées
- l'adaptation progressive au style de l'utilisateur

## Build APK
Quand du code est poussé sur `main`, GitHub Actions lance un build debug.
Tu pourras récupérer l'APK depuis l'onglet **Actions** du repo.
